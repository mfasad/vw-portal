MCC-коды для Яндекс Заправок: полное руководство 2026

Если вы владелец автозаправочной станции или занимаетесь добавлением точек на Яндекс Карты, то рано или поздно столкнётесь с понятием MCC-кода. Этот четырёхзначный идентификатор определяет тип бизнеса и критически важен для корректного отображения заправки в сервисах Яндекса — от поиска до навигации. Без правильного MCC ваша АЗС может не попасть в категорию "Заправки", потерять клиентов или даже быть удалена из базы.

В этой статье мы разберём, какие MCC-коды использует Яндекс для заправок, как их проверить, почему они отличаются от банковских классификаторов, и что делать, если ваша станция не определяется корректно. Также вы узнаете о нюансах работы с Яндекс Бизнесом и API, которые напрямую зависят от этих кодов.

Что такое MCC-код и почему он важен для заправок

MCC (Merchant Category Code) — это стандартный классификатор типов бизнеса, который изначально разработали платежные системы (Visa, Mastercard) для идентификации транзакций. Однако Яндекс использует свою адаптированную версию этих кодов для категоризации объектов на картах. Для заправок это означает:

  • 🔹 Поисковая выдача: без правильного MCC ваша АЗС не появится в результатах по запросу "заправка рядом".
  • 🔹 Навигация: Яндекс Навигатор и Карты фильтруют точки по MCC при построении маршрутов с остановками на АЗС.
  • 🔹 Реклама: таргетированные объявления в Яндекс Директе для водителей работают на основе этих кодов.
  • 🔹 API и партнёрские сервисы: агрегаторы топливных цен (например, PetrolMap) pulls данные по MCC.

Важно понимать, что Яндекс использует собственную базу MCC, которая не всегда совпадает с банковскими стандартами. Например, код 5541 (стандартный для АЗС в платежных системах) в Яндексе может дробиться на подкатегории или заменяться на внутренние идентификаторы.

Основные MCC-коды для заправок в Яндексе

Вот актуальные коды, которые Яндекс использует для классификации автозаправочных станций в 2026 году:

MCC-код Описание категории Примеры объектов
5541 Стандартная автозаправочная станция (АЗС) Лукойл, Газпромнефть, Роснефть
5542 Автоматические заправки (без кассира) Shell Recharge, Татнефть-Автомат
5983 Топливные карты и сервисные центры Пункты продажи карт Фуел Кард, Дривво
7538 Автосервисы при АЗС Шиномонтаж, мойки, СТО на территории заправки

Если ваша заправка комбинированная (например, с кафе или магазином), Яндекс может назначить дополнительные коды. Например, для АЗС с мини-маркетом часто используют комбинацию 5541 (заправка) + 5411 (супермаркет).

⚠️ Внимание: Если ваша станция имеет код 5999 (прочие специализированные магазины), она не будет отображаться как заправка. Это распространённая ошибка при ручном добавлении объектов.

Как проверить MCC-код своей заправки

Есть три надёжных способа узнать текущий MCC вашей АЗС в Яндекс Картах:

  1. Через Яндекс Бизнес:

    Авторизуйтесь в личном кабинете, выберите свою заправку и перейдите в раздел Информация о компании → Категории. Там будет указан основной MCC.

  2. Через API Яндекс Карт:

    Используйте запрос к Geocoder API с параметром kind=house и координатами вашей АЗС. В ответе ищите поле category.

    https://geocode-maps.yandex.ru/1.x/?apikey=ВАШ_API_КЛЮЧ&geocode=ДОЛГОТА,ШИРОТА&kind=house&format=json
  3. Через банковскую выписку:

    Если у вас подключён эквайринг, посмотрите MCC в выписке по транзакциям. Но помните: банковский код может отличаться от яндексовского!

Частые ошибки с MCC-кодами и как их исправить

Даже опытные менеджеры допускают ошибки при работе с MCC. Вот самые распространённые проблемы и их решения:

  • 🚨 Заправка не отображается в поиске:

    Причина: назначен код 5999 или отсутствует категория. Решение: в Яндекс Бизнесе установите 5541 как основной.

  • 🚨 АЗС показывается как магазин:

    Причина: основной код — 5411 (супермаркет). Решение: поменяйте приоритет категорий, поставив 5541 на первое место.

  • 🚨 Не работает фильтр "Заправки" в Навигаторе:

    Причина: отсутствует геометрия объекта (полигон территории АЗС). Решение: в Яндекс Картах отредактируйте границы станции через Режим редактирования.

  • 🚨 Дублирующиеся объекты:

    Причина: одна заправка добавлена несколько раз с разными MCC. Решение: объедините дубли через Яндекс Бизнес (раздел Дубликаты).

⚠️ Внимание: Если ваша заправка находится на территории торгового центра или рядом с крупным объектом (например, гипермаркетом), Яндекс может автоматически назначить неверный MCC. В этом случае обязательно уточните категорию вручную.
💡

Перед изменением MCC-кода проверьте, не используется ли он в ваших рекламных кампаниях в Яндекс Директе. Смена кода может сбросить статистику таргетинга.

MCC-коды для электрозарядных станций (EV)

С ростом популярности электромобилей Яндекс ввёл отдельные MCC-коды для зарядных станций. Они отличаются от традиционных АЗС:

MCC-код Тип зарядной станции Примеры сетей
5552 Автомобильные зарядные станции (общий код) Fastned, Tesla Supercharger
5551 Быстрые зарядки (от 50 кВт) Shell Recharge, BP Pulse
5542 Гибридные станции (топливо + электричество) Газпромнефть с зарядками Plug'n Drive

Если ваша станция поддерживает оба типа заправок (бензин/дизель + электричество), используйте комбинацию 5541 (АЗС) + 5552 (зарядка). Это позволит отображаться в обоих фильтрах.

Как добавить зарядную станцию в Яндекс Карты?

Для добавления новой EV-станции перейдите в Яндекс Карты, нажмите Добавить место и выберите категорию "Зарядная станция". Укажите оператора (например, Tesla), тип разъёмов (CCS, CHAdeMO) и мощность. После модерации объект появится на карте в течение 1–3 дней.

Как MCC-коды влияют на рекламу и аналитику

Корректный MCC-код напрямую влияет на эффективность рекламных кампаний и сбор статистики:

  • 📊 Яндекс Директ:

    При настройке таргетинга на водителей используйте сегмент "Посетители АЗС", который формируется по MCC 5541 и 5542.

  • 📈 Яндекс Метрика:

    В отчётах по источникам трафика можно фильтровать пользователей, посещавших заправки, по коду категории.

  • 💰 Партнёрские программы:

    Сервисы кэшбэка (например, СберСпасибо) часто привязывают бонусы к транзакциям с определёнными MCC. Для АЗС это обычно 5541.

Если вы запускаете рекламу для заправки, обязательно проверьте, что в настройках аудитории указаны правильные коды. Например, при таргетинге на владельцев электромобилей используйте 5552, а не 5541.

💡

Неправильный MCC-код может снизить CTR рекламы на 30–40%, так как объявления не будут показываться целевой аудитории (водителям).

API и автоматизация работы с MCC-кодами

Для разработчиков и владельцев сетей АЗС Яндекс предоставляет инструменты для работы с MCC через API:

  • 🔧 Geocoder API:

    Позволяет получить MCC по координатам или адресу. Пример запроса: 

    https://geocode-maps.yandex.ru/1.x/?apikey=ВАШ_КЛЮЧ&geocode=37.6176,55.7558&kind=house&format=json

    В ответе ищите поле "Category" → "code".

  • 🔧 Business API:

    Через него можно массово обновлять MCC для нескольких объектов. Документация: yandex.ru/dev/business.

  • 🔧 Static API:

    Полезен для отображения АЗС на собственном сайте с фильтрацией по MCC. Параметр ll=ДОЛГОТА,ШИРОТА&spn=0.01,0.01&l=map,skl&pt=ДОЛГОТА,ШИРОТА,pm2blm~5541 подсветит все заправки рядом.

Для автоматизации проверки MCC можно написать скрипт на Python, который будет парсить ответы API и сравнивать коды с эталонными значениями. Пример кода для проверки:

import requests


api_key = "ВАШ_API_КЛЮЧ"


latitude = 55.7558


longitude = 37.6176



url = f"https://geocode-maps.yandex.ru/1.x/?apikey={api_key}&geocode={longitude},{latitude}&kind=house&format=json"


response = requests.get(url).json()



mcc_code = response["response"]["GeoObjectCollection"]["featureMember"][0]["GeoObject"]["metaDataProperty"]["GeocoderMetaData"]["Address"]["Components"][0]["kind"]


print(f"MCC-код объекта: {mcc_code}")

⚠️ Внимание: При массовых запросах к API Яндекса используйте кэширование и соблюдайте лимиты (1 000 запросов в сутки для бесплатного тарифа). Превышение может привести к блокировке ключа.

FAQ: Частые вопросы о MCC-кодах для заправок

Можно ли использовать один MCC-код для нескольких заправок?

Да, если все станции относятся к одному типу (например, только традиционные АЗС с кодом 5541). Однако для гибридных станций (топливо + электричество) лучше указывать оба кода.

Что делать, если Яндекс автоматически назначил неверный MCC?

Нужно вручную отредактировать категорию в Яндекс Бизнесе. Если изменения не применяются, обратитесь в поддержку с доказательствами (фото вывески, документы о регистрации АЗС).

Как часто обновляются MCC-коды в Яндексе?

Яндекс обновляет классификатор раз в 1–2 года, но изменения для конкретных объектов могут применяться оперативно (в течение недели после редактирования).

Можно ли использовать банковский MCC вместо яндексовского?

Нет, это разные системы. Банковский MCC (5541) может не совпадать с внутренней классификацией Яндекса. Всегда проверяйте код через Яндекс Бизнес.

Как добавить новую категорию MCC, если её нет в списке?

Самостоятельно добавить категорию нельзя, но можно отправить запрос в поддержку Яндекс Карт с обоснованием (например, для инновационных типов АЗС).
