Что такое FCC Car Launcher и зачем он нужен?

FCC Car Launcher — это специализированный лаунчер для автомобильных головных устройств на базе Android, который заменяет стандартный интерфейс магнитолы на более функциональный. В отличие от обычного Android Auto, он позволяет:

  • 📱 Запускать любые приложения (включая YouTube, Netflix, игры) напрямую с экрана, без привязки к смартфону.
  • 🎨 Настраивать адаптивные темы под дневной/ночной режим, изменять расположение иконок и виджетов.
  • 🔄 Использовать жесты (свайпы, двойное касание) для быстрого доступа к функциям.
  • 🔊 Интегрировать голосовые помощники (Google Assistant, Алиса) с поддержкой офлайн-команд.
  • 🚗 Поддерживать мультиоконность (например, навигатор + музыка одновременно).

Основное преимущество перед стандартным Android Auto — отсутствие привязки к телефону. Все приложения работают непосредственно на головном устройстве, что устраняет задержки и проблемы с подключением. Однако есть и минусы: не все версии лаунчера совместимы с прошивками магнитол, а некоторые функции (например, Wireless Android Auto) могут требовать дополнительных патчей.

⚠️ Внимание: FCC Car Launcher не является официальным ПО от производителей автомобилей. Его установка может привести к потере гарантии на мультимедиа-систему, особенно если требуется разблокировка загрузчика или рут-права.

Способ 1: Установка без рут-прав (для большинства магнитол)

  1. Скопируйте файл fcc_launcher_v4.5.2.apk на SD-карту или внутреннюю память магнитолы.

  2. Откройте любой файловый менеджер (например, FX File Explorer) и запустите APK. Если установка заблокирована, разрешите установку из неизвестных источников в Настройки → Безопасность.

  3. После установки запустите лаунчер. Если он не становится основным, выполните в ADB: 

    adb shell pm grant com.fengcheci.launcher android.permission.WRITE_SECURE_SETTINGS

    adb shell am start -n com.fengcheci.launcher/.MainActivity

  4. Перезагрузите магнитолу. FCC должен запуститься автоматически.

Способ 2: Установка с заменой системного лаунчера (требует ADB)

Если стандартный способ не сработал, придётся заменить системный лаунчер через ADB:

  1. Подключите магнитолу к ПК и проверьте подключение: 

    adb devices

    Должно отобразиться устройство (например, RK3399:5555).

  2. Скачайте патч для системной установки и распакуйте его в папку с ADB.

  3. Выполните команды по очереди: 

    adb shell mount -o rw,remount /system

    adb push fcc_launcher.apk /system/priv-app/
    

    adb shell chmod 644 /system/priv-app/fcc_launcher.apk
    

    adb shell pm disable-user --user 0 com.android.launcher3

  4. Перезагрузите устройство командой adb reboot.

⚠️ Внимание: Неправильное использование ADB может привести к брику магнитолы (потере работоспособности). Если после перезагрузки система не стартует, попробуйте сбросить настройки через Recovery Mode (обычно зажатие Power + Volume Up при включении).
💡

Если после установки FCC лаунчер не запускается, проверьте, не блокирует ли его антивирус или встроенная защита Android (Play Protect). Отключите их временно.

Настройка FCC Car Launcher: оптимальные параметры

После установки лаунчер требует тонкой настройки для комфортного использования. Основные параметры:

1. Базовые настройки интерфейса

  • 📱 Размер иконок: Настройки → Внешний вид → Размер иконок (оптимально: 80–90%).
  • 🎨 Тема: Выберите Dark для ночной езды или Auto (автоматическое переключение).
  • 🔄 Жесты:
    • Свайп вверх — Открыть приложения.
    • Свайп влево — Предыдущее приложение.
    • Двойное касание — Голосовой помощник.

2. Оптимизация производительности

Для слабых магнитол (например, на RK3368) рекомендуется:

  • ⚡ Отключить анимацию: Настройки → Система → Анимация → Без анимации.
  • 🗑️ Очищать кэш раз в неделю: Настройки → Память → Очистить кэш.
  • 🔋 Отключить фоновые процессы для ненужных приложений.

3. Интеграция с Android Auto

Чтобы FCC работал параллельно с Android Auto:

  1. Установите Android Auto на магнитолу.
  2. В настройках FCC включите Режим совместимости → Android Auto.
  3. При подключении телефона выбирайте FCC как основной лаунчер.

Критическая особенность: на некоторых магнитолах (например, Dasaita Max10) при одновременной работе FCC и Android Auto может пропадать звук. Решение — отключить Bluetooth Audio в настройках Android Auto.

FAQ: Частые вопросы по FCC Car Launcher

🔍 Как вернуть стандартный лаунчер, если FCC не понравился?

Если FCC установлен как пользовательское приложение (без замены системных файлов), достаточно:

  1. Перейти в Настройки → Приложения.
  2. Найти FCC Car Launcher и нажать Отключить.
  3. Выбрать стандартный лаунчер в Настройки → Домой.

Если FCC установлен в /system, выполните в ADB:

adb shell pm enable com.android.launcher3

adb shell cmd package set-home-activity com.android.launcher3

📱 Можно ли использовать FCC на смартфоне (не в машине)?

Технически да, но это нецелесообразно. FCC оптимизирован для автомобильных экранов с крупными элементами управления и ограниченным вводом. На смартфоне он будет:

  • Занимать много места на экране (неэффективное использование пространства).
  • Медленно работать из-за отсутствия оптимизации для мобильных процессоров.
  • Конфликтовать с стандартными жестами Android (например, свайп вверх для дома).

Для смартфона лучше использовать специализированные лаунчеры типа Nova Launcher или Action Launcher.

🔄 Как обновить FCC до новой версии без потери настроек?

При обновлении через стандартную установку APK настройки сохранятся, но:

  1. Скачайте новую версию APK (проверьте совместимость с вашей магнитолой!).
  2. Установите поверх старой версии (не удаляйте старую!).
  3. Если после обновления лаунчер сбросился, восстановите настройки из бэкапа: 
    adb pull /data/data/com.fengcheci.launcher/shared_prefs/com.fengcheci.launcher_preferences.xml

Для модифицированных сборок перед обновлением обязательно сделайте бэкап папки /data/data/com.fengcheci.launcher.

🎵 Почему в FCC нет звука в YouTube или Spotify?

Проблема типична для магнитол с нестандартной аудиосистемой (например, Xtrons или Eonon). Решения:

  1. Проверьте вывод звука: в настройках приложения (YouTube/Spotify) выберите Медиа или Bluetooth (если подключён телефон).
  2. Установите патч: скачайте Audio Patch для FCC и прошейте через Recovery.
  3. Сбросьте настройки звука: в Настройки → Звук → Сброс.

Если ничего не помогает, проблема может быть в драйверах аудиочипа. В этом случае поможет только перепрошивка магнитолы.

Нет, FCC Car Launcher не поддерживает native-интеграцию с CarPlay. Однако есть обходные решения:

  1. Эмуляция через Android Auto: установите CarPlay2Air на магнитолу и подключите iPhone по Wi-Fi. FCC будет работать параллельно.
  2. Патч для dual-boot: некоторые прошивки (например, для Dasaita) позволяют переключаться между FCC и CarPlay через ADB:
    3. adb shell am start -a android.intent.action.MAIN -n com.apple.carplay/.CarPlayActivity

Полноценной замены CarPlay от FCC ждать не стоит — это принципиально разные протоколы (CarPlay использует iAP2, а FCC работает на базе Android).
